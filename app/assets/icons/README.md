# VPN client icons

Local UI assets for the public VPN connection modal.

Files are used only to identify compatible VPN/proxy clients in the connection UI.

Sources:
- hiddify.svg — official hiddify/hiddify-app assets/images/logo.svg
- v2rayng.png — official 2dust/v2rayNG Android launcher icon
- nekobox.png — official MatsuriDayo/NekoBoxForAndroid Android launcher icon
- happ.* — App Store artwork for Happ - Proxy Utility
- streisand.* — App Store artwork for Streisand

Do not use these icons as NeuroVPN branding.
